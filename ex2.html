HTML

To build a website, you should know HTML - the basic technology used to determine the structure of a website. HTML is used to determine whether your web content is identified as a paragraph, list, title, link, image, media player, form, or one of many other available components. or even the new element you define.

CSS

CSS (Cascading Style Sheets) is a declarative language that adjusts how a webpage will look in the browser. The browser attaches the CSS declaration type to the selected elements to display it correctly. A type of declaration includes the attributes and values of elements, which determine what the site will look like.

JavaScript

JavaScript is arguably more difficult to learn than related technologies such as HTML and CSS. Before attempting to learn JavaScript, you are strongly advised to get familiar with at least these two technologies first, and perhaps others as well.
